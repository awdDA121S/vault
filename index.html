Воскрешение в хардкоре
Умер в хардкоре? Не беда. Vault воскресит персонажа прямо на месте смерти - с вещами и опытом. Хардкор остаётся включённым, мир не удаляется.
Телепорт
Телепортация на любые координаты или к точке спавна - кровати или мировому спавну.
Восстановление
Полное восстановление HP и голода одной кнопкой.
Погода и время
Установить ясную погоду, сделать день или ночь прямо в файле сохранения.
Очистка эффектов
Убрать все негативные эффекты - яд, замедление, слепоту.
Починка предметов
Починить весь инвентарь и броню за один клик.
Бэкапы

NXBT-Bridge
3.2.5

Движок прямого патчинга NBT-структур. Полностью переработан алгоритм обхода compound-тегов, устранены утечки gzip-блоков, добавлена валидация целостности.

DataVersion 3953+≤12ms patch timeJava 1.21.11
// NXBT-Bridge 3.2.5 - core patch routine
const zlib = require('zlib'), fs = require('fs');
const MAGIC = Buffer.from('0a0000', 'hex');

async function patchLevelDat(filePath, fields) {
const raw = fs.readFileSync(filePath);
const buf = zlib.gunzipSync(raw);
if (!buf.slice(0,3).equals(MAGIC))
throw new Error('Invalid NBT header');

// Traverse compound tree, resolve field offsets
const tree = parseCompound(buf, 3);
const player = tree.Data.Player;
Object.assign(player, fields);

// Re-serialize, gzip, write back
const out = zlib.gzipSync(serializeNBT(tree));
fs.writeFileSync(filePath, out); // ~8ms
}
