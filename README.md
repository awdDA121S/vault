# Vault - Minecraft World Editor

Бесплатный редактор сохранений Minecraft Java Edition by Venth Team

## Скачать

**[Скачать Vault v2.0.4](https://github.com/awdDA121S/vault/releases/download/v2.0.4-mini/Vault.Setup.2.0.4.exe)**

## Возможности

- Воскрешение в хардкоре с сохранением вещей
- Включить / выключить хардкор
- Восстановить HP и голод
- Телепорт на координаты или к спавну
- Очистить эффекты
- Починить все предметы и броню
- Ясная погода / день / ночь
- Просмотр инвентаря
- Поиск и избранное для миров
- Статистика игрока
- Бэкапы с кастомным именем и восстановлением
- Поддержка мультиплеера
- 6 языков интерфейса

## Требования

- Windows 10/11 x64
- Minecraft Java Edition 1.21.11

## Установка

Скачай установщик с [releases](https://github.com/awdDA121S/vault/releases) и запусти его.

## Технология

NXBT-Bridge 3.3.5 - движок прямого патчинга NBT-структур (форк prismarine-nbt), DataVersion 3953+, ≤12ms patch time.
